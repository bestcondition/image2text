# 图片转文本

## 参数说明

- n: 每行字符数
- threshold: 二值化阈值
- image_inverse: 黑白转换

## 转换示例
```shell
# 安装依赖
pip install -r requirements.txt
```
```shell
# 运行
python cvt.py
```
